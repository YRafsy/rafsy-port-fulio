# rafsy-3d-portfulio

a serverless ai agent-creating-platform

## Getting Started

### Prerequisites
- Node.js 16.x or higher
- npm 8.x or higher

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

## Technologies Used
- Nodejs
- Nextjs
- Tailwind
- Typescript
